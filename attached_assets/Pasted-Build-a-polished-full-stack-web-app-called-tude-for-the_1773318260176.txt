CORE PRODUCT
Étude+ is a live learning platform where qualified professors in Tunisia can create paid live classes for students.
The most important feature in the whole product is the ability for a professor to create a class and create a live class session inside it with:
- a title
- a description
- a price in TND
- a duration in hours
- a date/time
- a join/start button
- an embedded video classroom experience

The app should support:
